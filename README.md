# suit_up_daily
내 옷장을 한눈에! <br>
옷장 관리, 코디가 가능한 어플 "Suit Up Daily" 의 소스코드 입니다.

# 구현한 기능

- 회원가입 & 로그인
- 이미지 편집
- 옷장 View
- 코디 스케치북
- 코디 공유 게시판

# 활용한 라이브러리

- Retrofit
- Glide
- Google Login
- Color Picker View
- Circle Indicator
